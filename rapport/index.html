Webbrapport Rasmus porthin

Specifikationen och planerande

Att planera webbsidan skenades till en början svår för att det var någonting jag inte hade gjort ordentligt förut. medan jag redan hade självständigt programmerat på en egen webbsida förut på egenhand. Så det tog en hell del tid mera en förväntat at få en bra plan färdig.

pecifikationen var något jag skev tidigt men ändå för sent in i kursen för att jag inte förstod riktigt hur mycket det var att göra på egenhand kom jag inte i gång så tidigt med själva arbetet.

Planeringarna som är wireframes och en site map gjorde jag med hjälp av drav io var det enlighet mig var lät at planera ut med olika element vad sajten skulle egentligen se ut som även om det tog sin tid att slutföra och att det inte skämdes som det roligaste saken att göra. För fonten använde jag google fonts och hittade jost som min vall och paletton som min color picker.

Uppsugande av sajten

Uppbunden av sajten till den största Dellen gick smidigt tak vare att jag redan hålit på med webbutveckling förut och lärt mig en hell de om CSS och html. Men en hel del nya saker kom emot under projekttiden som att sajten skulle vara responsiv som jag löste med att använda media queries. Som sen skapa ett problem med hur jag strkuturetat min navigation med en margin-left auto som jag löste med att i stälet använd flex box justify-content: flex-end och width: 100% för att se till att naven tar up hela åmrodet. Andra problem som jag stöter på under programmerandet k löste jag med at tita på hur det fungerade i ett exempel i guidar på webben och märka av vad som var fell skrivet eller vad som skulle fungera bättre. Sidorna jag använde för det mesta var w3 Scholls , css tricks websidor, länkarna ifrån powerpointara till de olika materialen. Men jag kopierade inte rakt någonting utan skrev och implementera det helt själ in till min sida de användes bara som exempel.

Editering av bilder och video för att passa webbsidan beter med tillexempel storleken inte var massiv eller att videon i inte var i 4k. Editeringen gjorde jag med gimp och avidemux som var nya för mej så jag fick lära mig en hell del hur man fick visa saker och henda och vad de var ifrån lektionerna i skolan även om denna del skenades tråkig .

Slutsats

Slutligen tog det …. Ungefär timar för projektet och jag lärde mig om flex box, bild – video editering och en hell del nya html css metoder som jag kan använda för att strukturera webbsidor i framtiden men märkte att det finns en hell del mera som man kan lära sig och förbättra och att bli vanare med att skriva kod så det går snabbare.
